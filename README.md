User Management App

A simple User Management application built using React and Vite.  
It allows users to add, edit, and delete user information.

Features 
- Add users with name and email  
- Edit existing users  
- Delete users  
- Simple and beginner-friendly UI  

Technologies Used 
- React  
- Vite  
- HTML & CSS
